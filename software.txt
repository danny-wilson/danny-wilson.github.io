		<p>Welcome to my software page. All programs are freely available to download.</p>

		<p>The harmonic mean <i>p</i>-value procedure (<a href="http://www.danielwilson.me.uk/abstracts/wilson_2019.html">Wilson 2019</a>) can be downloaded as an <a href="https://cran.r-project.org/package=harmonicmeanp">R package</a> and a development version is available on <a href="https://github.com/danny-wilson/harmonicmeanp/tree/dev">GitHub</a>. BASTA, a method for fast and accurate phylogeography described in <a href="http://journals.plos.org/plosgenetics/article?id=10.1371/journal.pgen.1005421">De Maio, Wu, O'Reilly and Wilson (2015)</a> can be downloaded as a <a href="https://bitbucket.org/nicofmay/basta-bayesian-structured-coalescent-approximation">package for BEAST2</a>. The coalescent methods for estimating parameters from SI, SIS and SIR epidemiological 
		models described in <a href="abstracts/dearlove_wilson_2013.html">Dearlove and Wilson (2013)</a> 
		can be downloaded as part of the latest version of <a href="http://code.google.com/p/beast-mcmc/">BEAST</a>. 
		Please follow the links for all other programs below.</p>
